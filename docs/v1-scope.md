## Goal
Launch a simple fashion ecommerce website to sell products via traffic from Facebook and Instagram ads.

## Actors
- Admin (platform owner)
- Seller (lists products)
- Customer (browses and purchases)

## In Scope (v1)
- Seller can add and manage products
- Customers can browse product listings
- Product detail page
- Add to cart
- Single-seller checkout
- Online payment
- Order creation and order status

## Out of Scope (Explicitly NOT in v1)
- Multi-vendor cart
- Coupons / discounts
- Reviews & ratings
- Wishlist
- Returns & refunds
- Cash on delivery (COD)
- Mobile apps (Android / iOS)

## Hard Constraints
- Mobile-first web only
- Monolithic application
- Low traffic assumptions
- No third-party marketplaces

## Non-Goals
- Advanced analytics
- Loyalty programs
- Personalization

## Scope Change Rule
Any change to this document requires:
- A new GitHub issue

- Explicit approval
